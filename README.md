# NAnDoroid Documentation

This repository contains the source code for the official [NAnDoroid-shell](https://github.com/na-ive/nandoroid-shell) documentation website. You can view the live site at [nandoroid.naive.my.id](https://nandoroid.naive.my.id).

> Built using the powerful combination of [Astro](https://astro.build) and [Starlight](https://starlight.astro.build/), and continuously deployed via GitHub Pages.

For details on how to contribute or run this project locally, please read our **[Development & Contribution Guide](https://nandoroid.naive.my.id/guide/development/)**.

*(Note: If the live site happens to be down, you can always browse the raw markdown files directly from `src/content/docs/` in this repository).*

## Technologies Used

- [Astro](https://docs.astro.build) - The web framework
- [Starlight](https://starlight.astro.build/) - The documentation theme
- [Expressive Code](https://expressive-code.com/) - For beautifully rendered code blocks
