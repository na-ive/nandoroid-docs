# NAnDoroid-shell

Desktop shell berbasis Quickshell untuk Hyprland dengan gaya desain ala Android 16.

> **Catatan**: Shell ini beserta seluruh dependensinya dirancang khusus hanya untuk **distro berbasis Arch Linux** (Arch, CachyOS, EndeavourOS, dll).

**Versi:** v1.4.0  
**Lisensi:** AGPL-3.0

## Fitur Utama

- **Universal Dynamic Island:** Menampilkan indikator media, perpindahan workspace, timer pomodoro, sampai notifikasi popup, semuanya menyatu di dalam satu notch di tengah layar.
- **Kustomisasi Mendalam:** Opsi personalisasi yang luas (jam, visual lockscreen, ukuran UI) bisa diakses langsung lewat panel Settings bawaan.
- **Warna Otomatis:** Seluruh tema shell akan beradaptasi secara dinamis mengikuti warna wallpaper kamu menggunakan token desain Material 3 (via Matugen).
- **Koleksi Wallpaper Online:** Cari dan pasang wallpaper dari Wallhaven maupun repositori GitHub pribadi secara instan langsung dari shell.
- **...dan Masih Banyak Lagi:** Jelajahi ekosistem mulus yang sudah dilengkapi dengan tool OCR bawaan, smart screen recording, system monitor intuitif, serta berbagai interaksi mikro yang diracik buat ningkatin pengalaman desktop kamu.
